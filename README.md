## Android Mock Location Helper (AMLH)

AMLH is a tool sends mock GPS location to Android emulator.

## Requirement

  * Ruby 1.9.3+
  * Sinatra

## Usage

    git clone https://github.com/wancw/android-mock-location-helper.git amlh
    cd amlh
    bundle install
    rackup
    # open http://localhost:9292/
